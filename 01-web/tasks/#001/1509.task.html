Link and meta tags

En los codigos basicos que hemos creado en esta primera tarea hemos abordado los temas de Etiquetas y los atributos que pueden tener estas etiquetas. Las ultimas dos etiquetas que veremos seran las de link y meta, las cuales sirven como parte informativa para quien lea el codigo, ya que estas no generaran un cambio en la pagina web. teniendo esto en cuenta estas etiquetas se dan para dar informacion adicional como el tipo de lenguaje utilizado, el autor del codigo, la ubicacion del archivo, el tañaño de las imagenes utilizadas, entre otros. Ademas de dar informacion al lectro del codigo tambien se le suministra informacion al motor de busqueda en el cual se despliegue el codigo, llamados metadatos estos son invisibles para le usuerio de lapagina web ya que su finalidad es ayudar a la gestion del contenido de la pagina, estas meta tags siguen una costruccion muy parecida y ademas de los datos anteriormente nombrados tambien se pueden utilizar para darle mas detalle a la pagina web como la codificacion de caracteres, un ejemplo de esto puede ser para que no muestre un erro con la letra "Ñ". tambien se utilizan para referir al autor y darle derechos al codigo, evitando el copyright. Tambine sirven para darle istrucciones a los boots o roborts de busqueda. Tambien pueden servir como control de cache y para recargar la pagina. Todas estas meta tags deben ser utilizadas de acuerdo a la finalidad que tenga la pagina creada para asi lograr no solo un aspecto estetico si no ademas un buen funcionamiento de la misma.
